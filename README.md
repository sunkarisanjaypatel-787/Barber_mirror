# Barber's Mirror
**Biometric Vision Engine for Real-Time Analysis**

Barber's Mirror is a computer vision project focused on high-precision facial geometry analysis. By leveraging Euclidean mathematical mapping, this project bypasses the need for massive, compute-heavy Convolutional Neural Network (CNN) datasets, achieving high-accuracy detection with minimal resource overhead.

## Core Objectives
- **Geometric Mapping:** Utilizing MediaPipe to isolate facial landmarks and generate real-time topographical data.
- **Mathematical Optimization:** Implementation of Euclidean distance formulas to categorize face shapes, ensuring accuracy across diverse facial structures without reliance on pre-trained black-box models.
- **Resource Efficiency:** Designed to run efficiently on mobile hardware and edge devices.
